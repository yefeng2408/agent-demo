某个 claim 的状态演化路径（按时间）cypher：
MATCH (u:User {id:$uid})-[:HAS_TRANSITION]->(t:StatusTransition {claimKey:$claimKey})
RETURN t
ORDER BY t.at ASC

画“链路图”（t 节点串起来）
MATCH (u:User {id:$uid})-[:HAS_TRANSITION]->(t:StatusTransition {claimKey:$claimKey})
WITH t ORDER BY t.at ASC
WITH collect(t) AS ts
UNWIND range(0, size(ts)-2) AS i
WITH ts[i] AS a, ts[i+1] AS b
MERGE (a)-[:NEXT]->(b)
RETURN a,b

ORDER BY t.at DESC
LIMIT 50

我在没有 Kubernetes 的情况下，用 Docker Compose 实现了 Blue-Green Deployment，
通过 Nginx reload 实现 0 downtime 切换，同时保持 Sidecar LLM 推理和向量库一致性。

部署流程：
① 拉代码
    cd agent-stack
    git pull

② build 绿
    #docker compose up -d --build agent_green
    docker compose build agent_green
    docker compose up -d --build

③ 验证绿
    curl http://127.0.0.1:8082/actuator/health
    返回 "status":"UP"

④ 切流量
    ./deploy/switch.sh green

⑤ 留回滚
    如果发现新版本有问题
    执行 ./deploy/switch.sh blue
    直接秒回旧版本

chmod +x deploy/switch.sh
